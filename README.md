Projeto simples desenvolvido durante estudos de lógica de programação em JavaScript.

O objetivo é calcular o saldo mensal a partir de:

- salário total
- gastos fixos
- gastos no débito
- gastos no crédito
- valor destinado à poupança

## Funcionalidades

- soma automática de gastos por categoria
- cálculo do saldo final
- organização de despesas em arrays

## Tecnologias utilizadas

- JavaScript
